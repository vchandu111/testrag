## üöÄ Getting Started

### Recommended Learning Path

1. **Start with Project 1** - Most comprehensive guide with complete code
2. **Try Project 2** - Learn about citations and metadata
3. **Choose based on interest** - Pick projects that match your goals

### Prerequisites (Common to All Projects)

- Python 3.8 or higher
- Basic Python programming knowledge
- OpenAI API account (or alternative LLM provider)
- Understanding of basic RAG concepts (covered in Project 1)

### Quick Setup

```bash
# Create virtual environment
python3 -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install common dependencies
pip install langchain openai chromadb sentence-transformers pypdf2 streamlit python-dotenv
```
